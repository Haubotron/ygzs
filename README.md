# 小小蚁国辅助功能

## 项目介绍
这个项目提供了《小小蚁国》游戏的一些辅助功能，帮助玩家更好地理解和计划游戏中的进化材料需求以及区域战的积分计算。

## 功能特点
- **进化材料查询**：快速查找进化过程中所需的各种材料，例如鱼骨。
- **区域战积分计算**：提供一周中每一天的区域战积分计算，帮助玩家优化战略和资源分配。

## 如何使用
1. 下载或克隆本项目到本地。
2. 根据需要查询的材料或计算积分，使用相应的功能。
